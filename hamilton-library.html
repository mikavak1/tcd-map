Hamilton Library

Departments: Science Library and lecture-theatre hub in the Hamilton Building (East End campus)

Hours: Typical published timetable (updated 23 June 2025): Mon-Fri 9:00 am-10:00 pm, Sat 9:30 am-4:00 pm, Sun Closed

General Information

College Maps describes Hamilton Library as part of the Hamilton Building, one of Trinity's East End developments on the Westland Row side. The building connects the O'Reilly Institute to the wider East End campus and combines lecture theatres with the main science library.

Accessibility Highlights

College Maps marks the Hamilton Building and Hamilton Library as accessible at all levels. For detailed access planning (routes, lifts, toilets, and event support), users should cross-check Trinity's latest accessibility guidance before visiting.

Facilities & Student Services

Hamilton functions as a combined teaching and study zone, with science library spaces plus nearby lecture theatres in the same building cluster. Library users should use the official opening-hours page for up-to-date opening and service desk windows during term, vacation, and exam periods.
